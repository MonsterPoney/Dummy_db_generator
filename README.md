# Dummy_db_generator
Generate an SQL request to add dummy values to a table and optionally create the table.

## Features
Any generated text is a lorem ipsum,
exception for varchar that can be a first name, last name or an email.

## Types

### Numbers :
	TINYINT
	SMALLINT
	MEDIUMINT
	INT
	BIGINT
	FLOAT
	DOUBLE
	DECIMAL
### Text :
	CHAR
	VARCHAR
	TEXT
	TINYTEXT
	MEDIUMTEXT
	LONGTEXT
### Dates :
	DATETIME
	DATE
	TIMESTAMP
	TIME
### Special :
	FirstName (VARCHAR)
	LastName (VARCHAR)
	Email (VARCHAR)
